# Overview

This repository provides the implementation artifacts and execution instructions for three proof-of-concept use cases that illustrate different federation strategies for digital twins in agriculture.

This repository is associated with the Engineering of Digital Twins conference (EDTconf) paper titled *"Characterizing Federated Digital Twins through Heterogeneity, Autonomy, and Distribution"*.

## Repository Structure

```text
.
├── Use_Case_1_Centralized_Federation
├── Use_Case_2_Hierarchical_Federation
└── Use_Case_3_Peer_to_Peer_Federation
```

## Use Cases

Each folder contains its own `README.md` file with specific instructions on how to build, run, and test the corresponding use case.

* [Use Case 1: Centralized Federation](./Use_Case_1_Centralized_Federation/README.md)
* [Use Case 2: Hierarchical Federation](./Use_Case_2_Hierarchical_Federation/README.md)
